# Lumibase Extensions

Extensions sub-repository for [Lumibase CMS](https://github.com/khuepm2).

This repo contains custom extensions, plugins, and integrations that extend the core functionality of the Lumibase CMS platform.

## Structure

```
extensions/
├── README.md
└── <extension-name>/     # Each extension lives in its own directory
```

## Getting Started

Clone this repo into your Lumibase workspace:

```bash
git clone <repo-url> extensions
```

Each extension follows its own setup instructions found in its subdirectory.

## Contributing

1. Create a new directory for your extension
2. Include a `README.md` with usage instructions
3. Open a pull request against `main`
